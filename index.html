This page demonstrates how CSS selectors work.

id

ids can be applied to only one item on a page

Here's how we hook an id attribute to an element:

id="id-div"

Here's how the CSS is applied to the element:

div#id-div{
    background-color:lavenderblush;
}

Inside my-div
class

class can be applied to more than one item on a page

Here's how we hook a class attribute to an element:

class="my-list"

Here's how the CSS is applied to the element:

li.my-list{
    background-color:lightgreen;

compound

A compound selector allows us to use multiple selectors separated by commas. In this way we can apply the same rules to several selectors.

Here's how the CSS is applied to the element:

div#new-div, li.new-list{
             background-color:lightblue;
}

Here's what it looks like:

I'm inside a div with an id of new-div

child

A child selector allows us to access the next immediate elements inside the current elements.

Here's how the CSS is applied to the element:

ul.my-kids > li {
             background-color:lightyellow;
}

Here's what it looks like:

  • I'm in a li inside a ul with a class of my-kids
  • I'm in a li inside a ul with a class of my-kids
  • I'm in a li inside a ul with a class of my-kids
descendant

Using a descendant selector, we can access all elements at any depth within the current element.

Here's how the CSS is applied to the element:

div.grandkids li {
             background-color:lightcyan;
}

Here's what it looks like:

  • I'm inside of an li that is a descendant of a div with a class of grandkids
  • I'm inside of an li that is a descendant of a div with a class of grandkids
  • I'm inside of an li that is a descendant of a div with a class of grandkids
multiple classes

We can apply multiple classes to an element by placing class names separated by spaces in the class attribute.

Here's how we hook multiple classes to an element:

class="add-border add-background"

Here's how the CSS is applied to the element:

.add-border{
            border:1px solid black;
}
.add-background{
    background-color:lightgrey;
}

Here's what it looks like:

  • I'm in a li with has two classes named "add-border add-background".
